# 教材电子版如何纠错？

### 新版纠错方法：

发现教材中的文字错误，可以点击页面上的铅笔图标。

![](img/jiucuo.png)

此时会进入教材编辑页面（需要注册）。

找到错误处，修改并提交即可。

![](img/jiucuo2.png)

我们会看到您提交的修改请求，审核无误后，会进行确认修改。



下述老版纠错方法效率较低，已经停止使用。
