### 五、八会穴的应用

八会穴是指脏、腑、气、血、筋、脉、骨、髓等精气所会聚的腧穴。八会穴首载于《难经·四十五难》：“腑会太仓（[中脘](https://www.gmzyjc.com/read/zjs/zjs3.2.1-0.1.1.3.11.md)），脏会季胁（[章门](https://www.gmzyjc.com/read/zjs/zjs3.1.9-12-0.0.4.3.13.md)），筋会[阳陵泉](https://www.gmzyjc.com/read/zjs/zjs3.1.9-12-0.0.3.3.34.md)，髓会绝骨，血会[膈俞](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.1.3.17.md)，骨会[大杼](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.1.3.11.md)，脉会[太渊](https://www.gmzyjc.com/read/zjs/zjs3.1.1-3-0.1.1.3.9.md)，气会三焦外一筋直两乳内（[膻中](https://www.gmzyjc.com/read/zjs/zjs3.2.1-0.1.1.3.16.md)）也。”

八会穴与其所属的八种脏器组织的生理功能有着密切关系，因此在临床应用时，每穴均治疗有关的组织、脏腑的病症。如筋病，取筋之会穴[阳陵泉](https://www.gmzyjc.com/read/zjs/zjs3.1.9-12-0.0.3.3.34.md)，因其位于膝下，膝为筋之府；脉病，取脉之会穴[太渊](https://www.gmzyjc.com/read/zjs/zjs3.1.1-3-0.1.1.3.9.md)，因其为手太阴经之原，居于寸口，为脉之大会；髓病，取髓之会穴绝骨，因其属于胆经，胆主骨所生病，骨生髓之故；脏病，取脏之会穴[章门](https://www.gmzyjc.com/read/zjs/zjs3.1.9-12-0.0.4.3.13.md)，因五脏皆禀于脾，为脾之募穴；腑病，取腑之会穴[中脘](https://www.gmzyjc.com/read/zjs/zjs3.2.1-0.1.1.3.11.md)，因六腑皆禀于胃，为胃之募穴；气病，取气之会穴[膻中](https://www.gmzyjc.com/read/zjs/zjs3.2.1-0.1.1.3.16.md)，因其为宗气之所聚，为心包之募穴；血病，取血之会穴[膈俞](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.1.3.17.md)，因其位于心[肝俞](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.1.3.18.md)穴之间，心主血，肝藏血之故；骨病；取骨之会穴[大杼](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.1.3.11.md)，因其近于椎骨（柱骨之根）之故。另外《难经·四十五难》又说：“热病在内者，取其会之[气穴](https://www.gmzyjc.com/read/zjs/zjs3.1.7-8-0.0.2.3.13.md)也”，说明八会穴还能治某些热性病。八会穴如表21。
