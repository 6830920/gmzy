### 二、妇科症状

妇科病症主要与任、冲二脉有关，治疗应从调其二脉入手，任脉自有专穴，冲脉则合于足少阴，所以多取任脉、足少阴经穴。另外，妇科疾病多属血分，故常配足厥阴、足太阴经穴。
