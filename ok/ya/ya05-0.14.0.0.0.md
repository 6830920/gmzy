## 5.14 周小农治消渴案

任某。气忿成病，乡医投以熟地、附片、绵耆、[党参](https://www.gmzyjc.com/read/bc/bc17-0.1.2.0.0.md)、故纸、益智、[首乌](https://www.gmzyjc.com/read/bc/bc17-0.3.2.0.0.md)、枣仁，不效。闰月十三日诊：因气心火下移，溲多不禁，有澄白①。耐则自遗，间日寒热，大便自利。症属肝热，肾气不坚，下消须防②。[白薇](https://www.gmzyjc.com/read/bc/bc03-0.5.2.0.0.md)、[白芍](https://www.gmzyjc.com/read/bc/bc17-0.3.4.0.0.md)、花粉、[柴胡](https://www.gmzyjc.com/read/bc/bc01-1.2.9.0.0.md)、丹皮、条芩、黑山栀、[萆薢](https://www.gmzyjc.com/read/bc/bc05-0.0.14.0.0.md)、石莲、[桑螵蛸](https://www.gmzyjc.com/read/bc/bc18-0.0.18.0.0.md)、[黄柏](https://www.gmzyjc.com/read/bc/bc03-0.2.3.0.0.md)、[覆盆子](https://www.gmzyjc.com/read/bc/bc18-0.0.17.0.0.md)、[鸡内金](https://www.gmzyjc.com/read/bc/bc14-0.0.5.0.0.md)、香连丸等，数剂自愈。

（《周小农医案》第163页）

〔注释〕

①澄白：小便有沉淀物。

②下消须防：必须提防其病演变为下消之证。

〔评按〕气忿成病，怎可妄用温补？溲多有澄白、寒热时作、大便自利，为肝经湿热而兼肾亏、虚中挟实之证，故治须清肝热、敛肾气，方不至成下消之变。
