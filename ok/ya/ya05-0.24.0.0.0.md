〔注释〕

①两候：指表邪未除，复传入里，

②损益：指加减变化。

〔评按〕本案邪已入里而表证未除，其热固可达之于表，其湿则非温不能化，治方中用附桂之类，即欲温化其湿浊而达热于表，可谓温病而用温法。
